# Major RB Properties Website

This is the official landing page for Major RB Properties, featuring:
- Interactive Hero Section with Typing Effect
- 3D Tilt Cards for Services
- Custom Cursor and Loading Screen
- WhatsApp Integration
- Gallery of Phases

## How to Deploy

### Option 1: Netlify Drop (Easiest)
1. Go to [Netlify Drop](https://app.netlify.com/drop).
2. Drag and drop the `MRB Properties` folder onto the page.
3. Your site will be live instantly!

### Option 2: GitHub Pages
1. Create a repository on GitHub.
2. Push this code to the repository.
3. Go to Settings > Pages and enable GitHub Pages from the `main` branch.
